# CLAUDE.md

## Design Context

This is NEBI, a Romanian brand selling modular solid-wood cat towers, donating 30% of profit to animal shelters. Static multi-page HTML/CSS/JS site (`index.html`, `magazin.html`, `colaborari.html`), pixel-art visual identity.

- **PRODUCT.md** — register, audience, positioning, brand personality, anti-references. Read before any strategic or copy decision.
- **DESIGN.md** — colors, typography, components, elevation, and the Do's/Don'ts derived from PRODUCT.md. Read before any visual change.

Both are maintained via the `impeccable` design skill (`/impeccable`).
